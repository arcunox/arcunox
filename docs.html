Arcunox Documentation

Arcunox is a deterministic email alias generator designed for privacy‑focused workflows. This documentation explains how alias generation works, how seeds behave, and how to use Arcunox for controlled identity segmentation without tracking, accounts, or analytics.

What Arcunox Generates

Arcunox produces email‑ready aliases based on a seed you provide. Aliases are stable, reusable, and predictable. They are not usernames, nicknames, or random strings.

How Alias Generation Works

  • Seed Input: email, domain, context string, or any custom identifier.
  • Deterministic Logic: inspectable, stateless computation with no hidden state.
  • Output: a clean alias suitable for sign‑ups, identity segmentation, and spam‑resistant workflows.

Use Cases

  • Creating unique aliases for online services
  • Separating personal, work, and project identities
  • Reducing spam with revocable aliases
  • Maintaining privacy without exposing your real inbox

What Arcunox Is Not

  • Not a username generator
  • Not a nickname tool
  • Not a random name generator
  • Not a tracking or analytics service

Privacy Notes

Arcunox stores nothing. All alias generation happens locally in the browser. No logs, no telemetry, no analytics, no accounts.
